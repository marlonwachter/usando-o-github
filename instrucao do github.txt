git init - iniciar
git status - verificar os arquivos atualizados, incluidos ou não

-Como Adicionar Arquivos no Controle de Versão-
git add "nome do arquivo usado" obs; pode ser que precisa do .(ponto)

-como criar versões do código com commit-
git commit -m "commit inicial"

-como enviar alterações usando git push-
git push  obs; define para aonde o cód vai se renviado
git remote add origin (copiar o link do github)

-verificar histórico da atualização do seu código-
git reflog

-como navegar entre versões do seu cód-
git reset --hard (digitar o codigo)

 ------Resumo-----

git init - inicializa um novo projeto
git add - adiciona os arquivos atuais ao proximo projeto
git status - verificar o status atual do repositório git
git commit -m "mensagem do commit" - cria um noco commit com uma memsagem
git push - envia as atualizações para a nuvem na branch atualmente ativa
git checkout - "nome da branch" - permite mudar para uma nova branch
git checkout -b "nome da branch de origem" "nome da nova branch" permite mudar e criar uma nova branch com base em outra
git merge - "branch a receber merge" - permite fazer o merge da branch ativa atualmente com outra branch
git pull - atualiza a branch atualmente ativa
